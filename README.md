# Waterphys

**Waterphys** is a small 2D physics/puzzle prototype built with **Godot** and **GDScript**.

The project experiments with simple water behavior, terrain digging, falling balls, and goal-based level logic.  
The player can carve channels into the terrain, release balls, add water, and observe how the simulated water mass spreads through the level.

This is not a finished game, but a playable prototype focused on interaction, simulation behavior, and puzzle mechanics.

## Status

**Prototype / playable experiment**

Waterphys is a small experimental project.  
It is useful as a portfolio piece for showing physics-style gameplay prototyping, Godot/GDScript usage, and interactive system design.

## Features

- Diggable terrain
- Simple 2D water simulation
- Falling ball objects
- Goal-fill detection
- Restartable levels
- Basic puzzle/prototype UI
- Mouse-based interaction

## Controls

- **Left click:** drop a ball
- **Shift + left click / middle mouse:** add water
- **Right mouse drag:** dig terrain
- **R:** restart level
