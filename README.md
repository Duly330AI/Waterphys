# Waterphys

A small 2D water-flow puzzle prototype built with Godot.

The player digs channels through the terrain to guide water into the bathtub.  
The project was created as an experiment around simple water behavior, tile interaction, and puzzle-style level goals.

## Status

Prototype / playable experiment. Not a finished game.

## Features

- Diggable terrain
- Simple water flow behavior
- Level goal detection
- Progress UI
- Restartable levels
